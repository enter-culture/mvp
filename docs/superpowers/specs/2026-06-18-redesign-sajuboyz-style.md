# 여행 추천 서비스 — sajuboyz 스타일 리디자인 설계 문서

## 개요

기존 단일 페이지 퀴즈 앱을 sajuboyz.com 스타일의 멀티 페이지 서비스로 전면 재설계.
배경 영상(Pexels 무료), 실제 이미지(Unsplash), 여행지 상세 페이지를 포함.

## 기술 스택

- Next.js 16 (App Router), TypeScript, Tailwind v4, FSD 아키텍처
- 영상: Pexels 무료 공개 영상 URL (직접 임베드)
- 이미지: Unsplash CDN URL

## 페이지 라우팅

```
/                    → 랜딩 페이지 (sajuboyz 스타일)
/quiz                → 퀴즈 3단계
/result              → 결과 + 페르소나 + 추천 카드
/destination/[id]    → 여행지 상세 페이지
```

### 3. 결과 페이지 `/result`

- 상단: 페르소나 타이틀 (대형 세리프)
- 서브 카피 + 스토리텔링 3~4줄
- 구분선
- "당신을 위한 여행지" 섹션

  - 카드: 이미지 70% + 지역명 + 제목
  - **클릭 → `/destination/[id]`**
- 하단: [다시 테스트] 버튼

## 페르소나 & 여행지 데이터 (더미)

### 페르소나 5종

1. **삼시세끼 노포 정착민** — 고즈넉한, 정겨운, 편안한 선택자
2. **파도 앞 멍때리기 선수** — 자유로운, 웅장한, 이국적인 선택자
3. **도심 감성 콜렉터** — 트렌디한, 인생샷, 팝업스토어 선택자
4. **산속 고독 탐험가** — 차분한, 웅장한, 트레킹 선택자
5. **로컬 미식 기록자** — 뿌듯한, 맛집탐방, 로컬미식 선택자

### 여행지 10곳 (각 페르소나당 2곳)

각 여행지:
- id, region, title, mood, description
- videoUrl (Pexels)
- images[] (Unsplash, 3장)
- spots[] (추천 스팟 3곳)
- tags[] (이런 분께 추천)
- personaId

## FSD 레이어 변경 사항

```
src/
├── app/
│   ├── page.tsx                    → 랜딩 (새로)
│   ├── quiz/page.tsx               → 퀴즈 (이동)
│   ├── result/page.tsx             → 결과 (새로)
│   └── destination/[id]/page.tsx  → 상세 (새로)
├── views/
│   ├── landing/ui/LandingPage.tsx  → 새로
│   ├── quiz/ui/QuizPage.tsx        → 기존 변환
│   ├── result/ui/ResultPage.tsx    → 새로
│   └── destination/ui/DestinationPage.tsx → 새로
├── widgets/
│   ├── video-hero/                 → 새로 (랜딩 + 상세 공용)
│   ├── chip-selector/              → 기존 quiz-screen 리팩터
│   ├── persona-card/               → 새로

├── entities/
│   ├── question/data/questions.ts  → 칩 전면 교체

│   └── destination/               → 새로 (10곳 데이터)
└── shared/ui/
    └── VideoBackground/            → 새로
```
